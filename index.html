About

I hold a Master's in Physics with Astronomy, driven by a fascination with the universe and the fundamental laws that describe it, in particular general relativity. During my studies, I developed a strong interest in programming through coursework and self directed projects, where I explored topics such as multithreading and version control with Git. My involvement in a high performance computing and optimization club further deepened my enthusiasm for large-scale software systems. This led me to pursue a second Master's in Scientific Computing and Data Analysis, which strengthened both my technical expertise and my passion for designing smart, scalable, and maintainable software solutions.

Projects

DSL for ExaHyPE 2

The project involved writing a DSL leveraging an embedded Python syntax to automatically generate optimised C++/SYCL kernels for high-performance simulation workloads in ExaHyPE 2, eliminating manual kernel programming. I collaborated heavily with EPCC's xDSL team, leveraging their cutting edge compiler optimisation technology to ensure scalable low-latency performance on distributed systems.

Fluctuation Eigenspectroscopy Analysis for JWST

Simulated galactic spectra that would be detectable by the James Webb telescope. Created a program that calculates Eigenspectra using the novel technique of Fluctuation Eigenspectroscopy. Used parallelization technologies and algorithm optimisation to efficiently run the program. Proved that a single 3 hour galaxy exposure could lead to scientifi c discoveries in TP-AGB stellar physics.
